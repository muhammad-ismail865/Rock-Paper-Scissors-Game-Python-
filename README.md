
# 🪨📄✂️ Rock Paper Scissors Game (Python) — GitHub Project

## 📌 1. Project Overview (for README)

This is a simple **Rock-Paper-Scissors game** built using Python.
The user plays against the computer, and the program randomly selects a move for the computer.

## ⚙️ 2. How the Game Works (Logic Explanation)

1. The user enters:

   * `r` = Rock
   * `p` = Paper
   * `s` = Scissors

2. Computer randomly selects one choice using:

   random.choice()
   
4. Game compares both choices:

   * If same → Tie
   * Otherwise → Winner is decided

5. User is asked if they want to continue:

   * `y` → play again
   * `n` → exit game

## Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
